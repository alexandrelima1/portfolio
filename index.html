Switch Access

Gestão de Conexões Cisco

Sistema desenvolvido para gestão de conexões em ativos de rede Cisco, simplificando tarefas de infraestrutura com funções predefinidas que evitam o uso de SSH e comandos shell. Utilizando Python, Netmiko, Paramiko, Tkinter e CustomTkinter, o software reduz erros e facilita a integração de novos colaboradores, melhorando o aprendizado e a segurança nas operações de rede.

Conex - Controle e Expedição

Gerenciador de Estoque

O Conex é uma solução prática e eficiente para controlar o estoque vigente, gerenciar entradas e saídas de equipamentos, componentes e periféricos. Com foco em organização e segurança, o sistema oferece:

  • Controle de Acesso de Usuários: Níveis de acesso personalizados para maior segurança e gestão de permissões.
  • Registros Detalhados: Logs de operações armazenados no banco de dados, garantindo rastreabilidade.
  • Relatórios e Validação: Geração de relatórios detalhados para análise e suporte à tomada de decisões.
  • Interface Intuitiva: Desenvolvido com Swing e aprimorado com FlatLaf, garantindo uma experiência visual moderna.

Criado em **Java** com banco de dados **PostgreSQL**, o sistema oferece confiabilidade e desempenho para pequenas e médias empresas.
